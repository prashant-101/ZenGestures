# ZenGestures
ZenGestures helps users practice yoga and hand exercises with live feedback using their webcam.
It detects body and hand movements using MediaPipe Pose and MediaPipe Hands, and provides guidance for better form and breathing.

🔑 Features
🧘 Yoga Pose Tracking – Recognizes and gives feedback on yoga asanas

✋ Hand Exercise Detection – Tracks ~10 hand flexibility & relaxation exercises

🌬 Breathing Guidance – Inhale/exhale animations for mindfulness

📊 Progress Dashboard – Daily streaks, exercise logs, and performance charts

⚡ Runs in Browser – No backend required (privacy-friendly)

🎨 Clean UI – Built with Next.js, Tailwind CSS, and Framer Motion

🚀 Tech Stack
Frontend: Next.js, Tailwind CSS, Framer Motion

Pose Detection: MediaPipe Pose, MediaPipe Hands

Charts & Tracking: Recharts
